#  📝 To-Do-List-Python-Program

A simple command-line To-Do List application built with Python.
This is my hands-on beginner project as part of my Data Science learning journey.

## 🚀 About This Project
A fully functional To-Do List program that runs in the terminal.
Built using core Python concepts — lists, functions, loops, file handling, and error handling.

## ✨ Features
- ✅ Add tasks
- 📋 View all tasks
- 🗑️ Delete tasks
- 💾 Save tasks to a file
- 📂 Load saved tasks on startup

## 🛠️ Built With
- Python 
- Jupyter Notebook
