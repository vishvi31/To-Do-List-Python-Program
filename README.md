## 📌 About

This is a simple **To-Do List program** written in Python using only core concepts —
no external libraries needed. It was built to practise Python fundamentals like
lists, functions, loops, conditionals, and file handling.

---

## ✨ Features

- ✅ Add tasks
- 📋 View all tasks (numbered list)
- 🗑️ Delete tasks by number
- 💾 Save tasks to a `.txt` file
- 📂 Load tasks from file on next session

---

## 🧠 Concepts Used

| Concept | Usage |
|---|---|
| Lists | Storing all tasks |
| Functions | `add_task()`, `view_tasks()`, `delete_task()` |
| for loop + enumerate | Displaying numbered tasks |
| if / elif / else | Validating user input |
| File handling | Save & load with `open()` |
| try / except | Handling missing file gracefully |
| f-strings | All output messages |
